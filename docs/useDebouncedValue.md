## About
Tracks another value and gets updated in a debounced way.


[//]: # (Main)

## Installation

```
npm install rooks
```

## Importing the hook

```javascript
import {useDebouncedValue} from "rooks"
```

## Usage

```jsx
function Demo() {
  useDebouncedValue();
  return null
}

render(<Demo/>)
```
